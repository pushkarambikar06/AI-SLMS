Composition is a strong relationship where one object owns another object.

COMMIT permanently saves database changes.

SYNTAX:
COMMIT;

EXAMPLE:
After successfully inserting records.

----------------------------------------

TOPIC: ROLLBACK

DEFINITION:
ROLLBACK undoes database changes if an error occurs.

SYNTAX:
ROLLBACK;

EXAMPLE:
Canceling a transaction after a failure.

----------------------------------------

TOPIC: DATABASE SECURITY

DEFINITION:
Database Security protects data from unauthorized access and threats.

EXAMPLE:
Restricting access to student records using user roles and passwords.
